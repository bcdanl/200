1.4 Question 3.

  • Write an R code to calculate the standard deviation (SD) of the integer vector x below manually. That is to calculate the SD without using the sd() function.
x <- 1:25
  • Also, write an R code to test whether the standard deviation you calculate manually above is equal to sd(x).

Answer:

numerator <- x - mean(x)
numerator <- numerator^2
numerator <- sum(numerator)
numerator
[1] 1300
denominator <- length(x) - 1
denominator
[1] 24
variance_x <- numerator/denominator
variance_x
[1] 54.16667
sd_x_manually <- sqrt(variance_x)
sd_x_manually
[1] 7.359801
sd_x <- sqrt( sum( (x - mean(x))^2 ) / (length(x) - 1) )
sd_x
[1] 7.359801
sd(x)
[1] 7.359801
sd_x == sd(x)
[1] TRUE
sd_x_manually == sd(x)
[1] TRUE

## Question 3.
- Write an R code to calculate the standard deviation (SD) of the integer vector `x` below manually. That is to calculate the SD without using the `sd()` function.

```{r}
x <- 1:25
```

- Also, write an R code to test whether the standard deviation you calculate manually above is equal to `sd(x)`.



*Answer*:
```{r}
numerator <- x - mean(x)
numerator <- numerator^2
numerator <- sum(numerator)
numerator
denominator <- length(x) - 1
denominator

variance_x <- numerator/denominator
variance_x
sd_x_manually <- sqrt(variance_x)
sd_x_manually


sd_x <- sqrt( sum( (x - mean(x))^2 ) / (length(x) - 1) )
sd_x

sd(x)

sd_x == sd(x)
sd_x_manually == sd(x)
```
